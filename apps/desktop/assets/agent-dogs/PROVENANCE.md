# Agent dog asset provenance

The five SVG prototypes in this directory were created from simple geometric shapes specifically for Harness on 2026-07-11. They do not contain, trace, modify, or derive from Agent Cat Connectors, its screenshots, or third-party sprite sheets. The Harness project dedicates these prototypes under CC0-1.0. Each uses the shared 128×128 logical canvas and the renderer supplies the state animation contract.
